10th Grade Java Assignments
==============================

This repository contains my 10th-grade Java programming assignments. 
It showcases a variety of projects, from basic console programs to loops, arrays, and small games.

------------------------------
Assignments
------------------------------

1. Assign 1 - Basic Console Apps
   Introductory console programs.

2. Assign 2.5 - Change for Twenty
   Problem-solving with conditionals.

3. Assign 3 - Loops
   Practice using for, while, and do-while loops.

4. Assign 4 - Arrays
   Array manipulation exercises.

5. Assign 5.1 - Hello World
   Beginner-level console output.

6. Assign 5.2 - Calculator
   Basic arithmetic calculator.

7. Assign 6 - Tic Tac Toe
   Console-based two-player game.

8. Currency Converter
   Converts between currencies.

9. InvestmentCalculator
    Simple investment projection.

10. Loop Test Study
    Loop practice exercises.

12. LoopHW
    Additional loop exercises.

13. Tax Calculator
    Calculates tax based on input.

------------------------------
Key Concepts Covered
------------------------------

- Console Input/Output
- Loops and Conditionals
- Arrays and Collections
- Methods and Functions
- Basic Object-Oriented Programming
- Simple Game Logic
